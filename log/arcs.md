# ARCS — the seasons of the legend

The campaign runs in named arcs, like the acts of a long tale. An arc's TITLE is written only when it ends; naming it early is a jinx and a railroad. Each closing arc gets an epilogue montage: one line for every NPC the arc touched, where the arc left them.
