# MERDOU

A persistent solo campaign. The files are the game.

- `CLAUDE.md` — the GM's brain. Prose law, contract, systems.
- `state/` — truth. HP, crew, bounty, ascension, world, clocks.
- `log/` — what happened.
- `news/` — what the papers said happened. Often wrong.
- `tools/roll.sh` — real dice.

Start: open in Claude Code and say `start session zero`.
