# MERDOU

A persistent solo campaign in the Forgotten Realms, 1495 DR. The files are the game.
One player (Mert), one character (Merdou), one ambition (a dragon god of war), coast-wide bounty rules, permanent consequences.

## The map of the repo
- `CLAUDE.md` — the GM's brain: prose law, naming law, lore law, the contract, every system
- `state/` — truth: the man, vows, bonds, threads, clocks, crew, bounty, jobs, world
- `lore/` — the real Realms: city, wild, gods, legends, factions, calendar, street life
- `gm/` — **the vault. Reading it spoils your own game. It has no lock.**
- `log/` — what happened, session by session
- `news/` — what the broadsheets claim happened. Wrong on purpose
- `tools/` — real dice: `roll.sh` (checks, `--dc`, advantage, hidden), `name.sh` (canon name banks), `oracle.sh` (yes/no odds, twists, scene tests), `wild.sh` (region encounters), `event.sh` (world beats)

## Playing
Open the repo in Claude Code (phone or desktop; git keeps them the same game) and say **start session**. The GM reads the state, rolls the world forward, recaps in three lines and hands you the scene. Say **end session** to close the books.

Nothing is real until it is written to a file. Everything written is permanent.
